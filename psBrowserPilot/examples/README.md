# psBrowserPilot フローサンプル

`examples` ディレクトリには psBrowserPilot / NodeFlow で作成したサンプルフロー (`*.psbrowser.json`) を配置します。
GUI の「Load」ボタンからインポートすることで、初心者ユーザーが入出力ポートの接続方法を理解できるようにします。

## サンプル構成のヒント
- 基本的なファイル操作、REST 呼び出し、Excel 操作など、業務で頻出するユースケースを想定します。
- 各ノードの役割をコメントノードや説明書きで補足すると再利用しやすくなります。
- セッションで読み込むモジュールが必要な場合は `psBrowserPilot-GUI.html` の設定欄に記載し、README にも追記してください。
